## OpenClaw Optimisation Research — 2026-04-21

### Skills vs Plugins

- **Plugins**: Code that runs in OpenClaw gateway itself, affecting how I operate. Examples: Active Memory, Telegram channel, minimax model.
- **ClawHub**: Central repo for skills. Install via `clawhub install <skill>`.
